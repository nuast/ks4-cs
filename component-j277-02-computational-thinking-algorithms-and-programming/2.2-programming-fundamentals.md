## Key terms

| Term | Definition |
|------|-----------|
| Variable | A named memory location that holds a value that can change |
| Constant | A named memory location that holds a value that cannot change during execution |
| Data type | The kind of data stored: integer, real, Boolean, character, string |
| Integer | A whole number (positive, negative, or zero) |
| Real (float) | A number with a decimal point |
| Boolean | A value that is either `true` or `false` |
| String | A sequence of characters |
| Sequence | Instructions executed one after another, in order |
| Selection | Choosing which block of code to run based on a condition (if/else/switch) |
| Iteration | Repeating a block of code (loops) |
| Array | A data structure storing multiple values of the same type under one name |
| Index | The position of an element in an array (starts at 0 in OCR ERL) |
| Procedure | A named sub-program that performs a task but does not return a value |
| Function | A named sub-program that performs a task and **returns** a value |
| Parameter | A variable passed into a sub-program when it is called |
| MOD | Returns the remainder of a division (e.g. `7 MOD 3 = 1`) |
| DIV | Returns the integer (whole number) part of a division (e.g. `7 DIV 3 = 2`) |

## Core notes

### Data types

| Type | Example values | ERL keyword |
|------|---------------|-------------|
| Integer | `0`, `42`, `-7` | `int` |
| Real | `3.14`, `-0.5`, `9.0` | `float` or `real` |
| Boolean | `true`, `false` | `bool` |
| Character | `'a'`, `'Z'`, `'3'` | `char` |
| String | `"hello"`, `"OCR"`, `"42"` | `str` or `string` |

> A string `"42"` is NOT the same as the integer `42` — they are different types.

### Variables and constants

```
// Variable – can change
score = 0
score = score + 10

// Constant – cannot change
const MAX_SCORE = 100
```

Use constants for values that should not change (e.g. tax rate, screen size, maximum array length).

### Sequence

Instructions run top to bottom, in order:

```
name = input("What is your name? ")
print("Hello, " + name)
print("Welcome to the quiz!")
```

### Selection

**if / else:**
```
score = int(input("Enter score: "))
if score >= 50 then
    print("Pass")
else
    print("Fail")
endif
```

**if / else if / else:**
```
if score >= 70 then
    print("Distinction")
elseif score >= 50 then
    print("Pass")
else
    print("Fail")
endif
```

**switch / case:**
```
day = input("Enter day number: ")
switch day:
    case "1":
        print("Monday")
    case "2":
        print("Tuesday")
    default:
        print("Invalid day")
endswitch
```

### Iteration

**Count-controlled (for loop):**
```
for i = 1 to 5
    print(i)
next i
```

**Condition-controlled (while loop):**
```
password = input("Enter password: ")
while password != "secret"
    print("Incorrect. Try again.")
    password = input("Enter password: ")
endwhile
```

**Do-while (repeat-until style):**
```
do
    number = int(input("Enter a positive number: "))
until number > 0
```

### Operators

**Arithmetic:**

| Operator | Meaning | Example | Result |
|----------|---------|---------|--------|
| `+` | Addition | `5 + 3` | `8` |
| `-` | Subtraction | `10 - 4` | `6` |
| `*` | Multiplication | `6 * 7` | `42` |
| `/` | Division | `10 / 4` | `2.5` |
| `MOD` | Remainder | `10 MOD 3` | `1` |
| `DIV` | Integer division | `10 DIV 3` | `3` |
| `^` | Exponentiation | `2 ^ 8` | `256` |

**Comparison (return Boolean):**
`==` (equal), `!=` (not equal), `<`, `>`, `<=`, `>=`

**Logical:**
`AND`, `OR`, `NOT`

### String manipulation

```
name = "Alice"

print(len(name))         // 5
print(name.upper)        // ALICE
print(name.lower)        // alice
print(name.substring(0, 3))  // Ali  (from index 0, length 3)
print("Hello, " + name)  // Hello, Alice  (concatenation)
```

> OCR ERL uses `len()`, `.upper`, `.lower`, `.substring(start, length)`.

### Arrays

```
// Declare and initialise a 1D array (5 elements, index 0 to 4)
array scores[5]
scores[0] = 72
scores[1] = 85
scores[2] = 60
scores[3] = 91
scores[4] = 78

// Access an element
print(scores[2])   // 60

// Loop through an array
for i = 0 to 4
    print(scores[i])
next i
```

### Sub-programs

**Procedure (no return value):**
```
procedure greet(name)
    print("Hello, " + name + "!")
endprocedure

greet("Alice")   // Hello, Alice!
```

**Function (returns a value):**
```
function square(n)
    return n * n
endfunction

result = square(7)
print(result)   // 49
```

### File handling

```
// Writing to a file
myFile = open("scores.txt")
myFile.writeLine("Alice: 85")
myFile.writeLine("Bob: 72")
myFile.close()

// Reading from a file
myFile = open("scores.txt")
while NOT myFile.endOfFile()
    print(myFile.readLine())
endwhile
myFile.close()
```

### SQL basics

```sql
SELECT name, score
FROM results
WHERE score >= 50
```

- `SELECT` – choose which columns to retrieve
- `FROM` – which table
- `WHERE` – filter rows by condition

## Worked examples

**Example 1 – Combining constructs: highest score finder**

```
array scores[5] = [72, 85, 60, 91, 78]
highest = scores[0]
for i = 1 to 4
    if scores[i] > highest then
        highest = scores[i]
    endif
next i
print("Highest score: " + str(highest))
```

Trace table:

| i | scores[i] | highest |
|---|-----------|---------|
| — | — | 72 |
| 1 | 85 | 85 |
| 2 | 60 | 85 |
| 3 | 91 | 91 |
| 4 | 78 | 91 |

Output: `Highest score: 91`

---

**Example 2 – Using MOD and DIV**

```
number = 17
print(number MOD 5)    // 2  (17 ÷ 5 = 3 remainder 2)
print(number DIV 5)    // 3  (17 ÷ 5 = 3, ignore remainder)
```

Useful: `number MOD 2 == 0` checks if a number is even.

## Common mistakes

- ❌ Confusing `=` (assignment) with `==` (comparison) – `score = 5` sets the value; `score == 5` tests whether the value is 5.
- ❌ Using a procedure where a function is needed (or vice versa) – procedures perform a task; functions return a value.
- ❌ Forgetting that array indices start at **0** in OCR ERL.
- ❌ Not closing files after reading/writing – always call `.close()`.
- ❌ Treating a string `"5"` as an integer `5` – always convert types explicitly if needed (`int("5")`).
- ❌ Using a `while` loop where a `for` loop is more appropriate, or vice versa – `for` when you know how many iterations; `while` when you don't.

## Quick-check questions

1. What is the difference between a variable and a constant?
2. What does `17 MOD 5` equal? What does `17 DIV 5` equal?
3. Write a `for` loop in OCR ERL that prints the numbers 1 to 10.
4. What is the difference between a procedure and a function?
5. An array is declared as `array colours[4]`. What are the valid indices?

<details>
<summary>Answers</summary>

1. A variable holds a value that can change during the program's execution; a constant holds a value that is set once and cannot change.
2. `17 MOD 5 = 2`; `17 DIV 5 = 3`.
3. `for i = 1 to 10` / `print(i)` / `next i`
4. A procedure performs a task but does not return a value; a function performs a task and returns a value to the calling code.
5. Indices 0, 1, 2, 3 (four elements, zero-indexed, so index 0 to 3).

</details>

**Q1.** State **two** differences between a variable and a constant. [2 marks]

*[Mark scheme: A variable can change its value during execution / a constant cannot (1); a constant is used for values that should remain fixed throughout the program (1).]*

**Q2.** Write an algorithm in OCR Exam Reference Language that asks the user to enter 5 numbers, stores them in an array, and then outputs all numbers greater than 10. [4 marks]

*[Mark scheme: Array declared (1); loop to input 5 values into array (1); loop to check each element (1); selection to output only values > 10 (1). Accept valid equivalent ERL.]*

---

**Q3.** The following pseudocode contains a logic error. Identify the error and write the corrected line. [2 marks]

```
for i = 1 to 10
    if i MOD 2 = 1 then
        print(str(i) + " is even")
    endif
next i
```

*[Mark scheme: Error: `MOD 2 = 1` identifies odd numbers, not even numbers (1); corrected: `if i MOD 2 == 0 then` (or equivalent) and change output to say "is even" (1).]*

---

**Q4.** Write a function in OCR ERL called `calcArea` that takes two parameters, `width` and `height`, and returns the area (width × height). [2 marks]

*[Mark scheme: `function calcArea(width, height)` (1); `return width * height` and `endfunction` (1).]*
