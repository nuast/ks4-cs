## Exam tips for J277/01

- Multiple-choice questions test recall – learn definitions precisely.
- Short-response questions often use **describe** or **explain** – practice writing focused answers.
- Extended-response questions (4–6+ marks) may ask you to **compare**, **discuss**, or **evaluate** – plan before you write.
- Show any calculations clearly with working (e.g. file size, binary conversions).
- Learn to **convert** between binary, denary, and hexadecimal confidently.
